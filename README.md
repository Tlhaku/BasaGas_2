# BasaGas

This repository delivers a minimal MEAN (MongoDB, Express, Angular, Node.js) starter for the BasaGas LPG refill platform. Part 1 of the software requirements is implemented in `basagas-mean/`, featuring a tiny Express API, MongoDB integration, and an Angular 17 front end that can be run locally on desktop or mobile browsers.

## Contents

- `basagas-mean/` — Self-contained MEAN template with server and Angular client code.
- `LICENSE` — MIT license for the project.

Refer to [`basagas-mean/README.md`](basagas-mean/README.md) for detailed setup and usage instructions, including required dependencies and how to run the API and UI locally.
